*************
Slitlessutils
*************

``Slitlessutils`` is a package for simulating and extracting wide-field
slitless spectroscopy (WFSS) data, highly focused on data from the
Advanced Camera for Surveys (ACS) and Wide-Field Camera 3 (WFC3) on the
*Hubble Space Telescope*.


.. important::
   If you use ``slitlessutils`` for a project that leads to a publication,
   whether directly or as a dependency of another package, then please
   include an `acknowledgment and/or citation <citation>`_.
